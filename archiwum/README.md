# Archiwum (nieaktywne komponenty)

Ten katalog zawiera elementy wyłączone z aktywnego rozwoju:

- `bridge/` — dawny mostek Python (WebSocket/Serial) i symulator.
- `hardware/arduino.ino` — historyczny firmware Arduino.
- `web/js/app/transport/ws.js` — stary transport WebSocket dla UI.

Aktualny tor danych projektu opiera się na `hardware/esp32.ino` + BLE UART + Web UI.
